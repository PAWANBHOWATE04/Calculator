# CodeAlpha Calculator

A simple, elegant web-based calculator built with HTML, CSS, and JavaScript. This project provides a user-friendly interface for performing basic arithmetic operations.

## Features

- **Basic Arithmetic Operations**: Supports addition (+), subtraction (-), multiplication (*), and division (/).
- **Decimal Support**: Includes a decimal point (.) for floating-point calculations.
- **Clear Functions**:
  - AC: Clears all input and results.
  - DE: Deletes the last entered character.
- **Zero Handling**: Includes buttons for 0 and 00 for quick input.
- **Real-time Display**: Shows the current expression and the calculated result separately.
- **Responsive Design**: Styled with a modern blue theme and subtle shadows for a clean look.

## Technologies Used

- **HTML**: Structure of the calculator interface.
- **CSS**: Styling for the layout, colors, and visual effects.
- **JavaScript**: Inline event handlers for button clicks and calculations using `eval()`.

## How to Use

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Click the buttons to input numbers and operators.
4. Press `=` to calculate the result.
5. Use `AC` to clear everything or `DE` to delete the last character.
